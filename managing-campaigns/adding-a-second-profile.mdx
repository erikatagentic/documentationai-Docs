---
title: "Adding a Second LinkedIn Profile"
---

Running outreach from a single LinkedIn profile works well for most clients. But there are situations where adding a second profile makes sense. Here's when and how.

## When It Makes Sense

**You've maxed out daily volume.** Each LinkedIn profile can safely send 25-35 connection requests per day. If you need more volume than that, a second profile doubles your reach without pushing either account past safe limits.

**You're targeting two different audiences.** If your business serves two distinct buyer personas (say, CFOs and HR Directors), running them from different sender profiles often performs better. Each profile can be optimized for its specific audience with a relevant headline and summary.

**You want a different sender persona.** Some clients run one profile for the founder and another for a sales rep. The founder profile tends to get higher acceptance rates (people like connecting with founders), while the sales profile handles higher volume.

## What Changes With Two Profiles

Each profile runs independently. They have separate:
- Daily sending limits
- Connection request queues
- Warmup schedules (if the second profile is new)
- Conversations in the Unibox (though you can see everything in one inbox with the sender filter)

Your campaign metrics are tracked per profile, so you can compare performance and see which sender is generating better results.

## How to Add One

Tell your account manager you'd like to add a second profile. We'll need:
- The LinkedIn credentials for the new sender account
- Whether it's an existing, active LinkedIn profile or a newer one (this affects warmup timeline)

We handle the rest: browser profile setup, proxy configuration, warmup, and campaign launch. The new profile typically takes 2-3 weeks to warm up before running at full volume.

## What About Overlap?

We make sure profiles don't contact the same prospects. The platform automatically deduplicates across sender accounts, so no one gets a connection request from both of your profiles.
